## Findings

### [F-001] Zero Coefficient Filtering (FIXED DURING DEV)

**Severity**: LOW
**Description**: `clamp(min=0)` crea coefficienti zero che `topk(k)` include anyway.
**Evidence**: Integration test mostrava coefficienti = 0 in output.
**Remediation**: Aggiunto filtro `if val > 0` in list comprehension per escludere zeri.
**Status**: ✅ FIXED

### [F-002] Vocabulary Dictionary Access (FIXED DURING DEV)

**Severity**: LOW
**Description**: `vocab_terms[idx]` restituisce dict invece di stringa (vocab è lista di dict con campi `term`, `similarity_score`, `source`).
**Evidence**: TypeError in integration test `"sequence item 0: expected str instance, dict found"`.
**Remediation**: Corretto a `vocab_terms[idx]["term"]` per estrarre il campo term.
**Status**: ✅ FIXED

### [F-003] Directory Creation Bug (FIXED DURING DEV)

**Severity**: LOW
**Description**: `ensure_dir(cfg.output_dir)` crea solo parent directories, non la directory stessa.
**Evidence**: `FileNotFoundError: [Errno 2] No such file or directory` durante test.
**Remediation**: Aggiunto `cfg.output_dir.mkdir(parents=True, exist_ok=True)` dopo `ensure_dir`.
**Status**: ✅ FIXED

### Key Achievements

1. **Determinismo totale**: Nessuna instabilità cross-seed (M-001 risolto per costruzione)
2. **Zero training**: CPU-only, <5 min per 1515 images
3. **Output compatibile**: Schema compatibile con SAE per downstream LLM judge

### Recommendations

1. **Coverage analysis**: k=32 è 3.1% del vocab (1030 termini). Monitorare coverage (% termini usati across test set) e aumentare k se <50% termini mai attivi.
2. **Lasso alternative**: Documentare Lasso(positive=True) come alternativa a OMP per true non-negative L1 (più lento ma coeff naturalmente non-negativi).
3. **LLM judge**: Valutare faithfulness su `results/spliece/sample_explanations.json` con MedGemma (post M-007 fixes).
4. **Modality gap impact**: Analizzare se gap correction migliora o peggiora decomposizione (test diagnostico già presente).
