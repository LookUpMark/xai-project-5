# Audits

This folder contains formal review and audit reports.

**What to include here:**
- ML pipeline audits (e.g., `ML-AUDIT-*.md`).
- Diagnoses of structural issues or bugs.
- Formal reviews of the methodology, architecture, or codebase.

**Usage:**
Use these documents to justify strategic pivots (like the post-audit reframe) or to document verified bugs and their resolution plans.
