# Design

This folder contains the core architectural and strategic documentation for the project.

**What to include here:**
- Active project strategy (`PROJECT-STRATEGY.md`).
- Active implementation plan (`IMPLEMENTATION-PLAN.md`).
- High-level architecture decisions.
- Historical or deprecated design notes (marked with a warning header).

**Usage:**
This is the single source of truth for *what* we are building and *how* we plan to build it. Always update the active strategy and plan documents here when the project direction changes.
