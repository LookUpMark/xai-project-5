# Proposals

This folder contains exploratory design documents, experiment proposals, and deep-dive diagnoses.

**What to include here:**
- Diagnoses of specific problems (e.g., `CONCEPT-INSTABILITY-DIAGNOSIS.md`).
- Proposals for new methods, ablations, or reframes (e.g., `PIPELINE-REFRAME-MAIN-VS-BASELINE.md`).
- Alternatives and brainstorming (e.g., `VOCAB-BUILDING-ALTERNATIVES.md`).

**Usage:**
Use this folder to propose and discuss major changes or experiments *before* they become the official strategy in the parent `design/` folder.
