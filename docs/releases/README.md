# Releases

This folder contains the history of project milestones and version releases.

**What to include here:**
- Changelogs (e.g., `CHANGELOG-V0-4-0-20260614.md`).
- Release notes summarizing major refactoring, feature additions, or bug fixes.

**Usage:**
Add a new markdown file here whenever a significant version bump or milestone is reached, detailing what changed, stats, and verification steps.
