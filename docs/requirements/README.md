# Requirements

This folder contains the core requirements and constraints for the project.

**What to include here:**
- The primary project brief (`PROJECT-BRIEF.md`).
- Evaluation and grading guidelines (`EVALUATION-GUIDELINES.md`).
- Constraints and requirements dictated by the course or stakeholders.

**Usage:**
Refer to these documents to ensure the project remains aligned with the required deliverables, evaluation rubrics, and research goals. Do not place implementation details here.
